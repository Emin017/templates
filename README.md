# Templates

## Usage

rust:
```shell
nix flake init -t 'github:Emin017/templates#rust'
```

scala:
```shell
nix flake init -t 'github:Emin017/templates#scala'
```

ocaml:
```shell
nix flake init -t 'github:Emin017/templates#ocaml'
```

riscv:
```shell
nix flake init -t 'github:Emin017/templates#riscv'
```
